The Quote Price Analysis uses the same Supabase Edge Function as the AI Plan
Estimator. Before testing the Quote Price Analysis, redeploy this file:

The existing Supabase secret must remain:

Optional model setting:

OPENAI_MODEL=gpt-5.6

The public API key must stay in Supabase secrets only. Never place the OpenAI
API key inside config.js, HTML or GitHub.

PROJECT STORAGE

Projects, tasks and saved results are stored on the current browser/device.
They remain after closing and reopening the link. Use Export Backup regularly,
especially before clearing browser data or changing devices. Restore Backup can
load the exported JSON file on another device.

This version does not automatically sync projects between devices. Automatic
multi-device sync requires a future Supabase login and database update.

QUOTE PRICE RULE

The $100 rule is applied separately to every matched quoted line:

- More than $100 above the matching AC line = EXPENSIVE
- More than $100 below the matching AC line = CHEAP
- Difference from -$100 through +$100 = FAIR
- No reliable AC match or missing line price = NEED REVIEW

The overall result is based on the item statuses:

- Expensive lines only = contains expensive items
- Cheap lines only = contains cheaper items
- Both expensive and cheap lines = mixed pricing
- Every matched line within $100 = fair pricing

The total difference is shown for information but does not decide the verdict.
